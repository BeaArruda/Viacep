# 📍 Consulta de CEP - ViaCEP

Projeto desenvolvido utilizando HTML, CSS e JavaScript para consumir a API pública ViaCEP e realizar consultas de endereço a partir de um CEP informado pelo usuário.

---

## 🚀 Funcionalidades

- Campo para digitar o CEP
- Botão para buscar o endereço
- Botão para limpar os dados
- Exibição formatada das informações:
  - CEP
  - Logradouro
  - Bairro
  - Cidade
  - Estado
  - DDD
- Validação de CEP inválido
- Tratamento de erro caso o CEP não exista

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Fetch API
- API pública ViaCEP

---

## 🔗 API utilizada

ViaCEP  
https://viacep.com.br/

Exemplo de requisição:
https://viacep.com.br/ws/01001000/json/

## ▶️ Como executar o projeto

1. Abra a pasta do projeto no VS Code
2. Instale a extensão **Live Server** (caso ainda não tenha)
3. Clique com o botão direito no arquivo `index.html`
4. Selecione **Open with Live Server**

O projeto será aberto no navegador através de um servidor local.
